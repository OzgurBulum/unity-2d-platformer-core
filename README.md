# Unity 2D Platformer Core

Unity 2D platformer mechanics learning project.

## Features
- Player movement with WASD and arrow keys
- Jump system with ground detection
- Variable jump height
- Pause menu system (Escape)
- Rigidbody2D physics-based movement

## Technical Details
- Unity Version: 6.3 LTS
- Platform: PC (Mac/Windows)
- Physics: 2D Rigidbody + Collider based

- **Move**: A/D or Left/Right Arrow Keys
- **Jump**: Space
- **Pause**: Escape
